# blinkit-sales-analysis-report-excel-dashboard
Excel-driven analysis of Blinkit sales data to understand customer demand patterns for Low Fat and Regular food products.
________________________________________
Project Objective:
To analyze Blinkit’s sales data using MS Excel only, with the goal of understanding how fat content in food items affects sales performance across different outlets and locations.
________________________________________
Data Description:
The dataset includes:
•	Item attributes: Item Type, Fat Content
•	Outlet attributes: Outlet Size, Outlet Location Tier, Outlet Type, Establishment Year
•	Performance metrics: Sales, Ratings, Number of Items
________________________________________
Workflow:
1.	Data Cleaning
o	Standardized fat content categories (Low Fat / Regular)
o	Validated numeric fields and removed inconsistencies
2.	Data Aggregation
o	Created Pivot Tables → sales by fat content
o	Analyzed performance across outlet size, location, and category
3.	Dashboard Development
o	KPI cards for Total Sales, Avg Sales, Item Count, Avg Rating
o	Donut charts showing fat-based sales contribution
o	Bar charts for outlet and category comparisons
o	Slicers for dynamic filtering
________________________________________
Key Findings:
•	Regular fat products contribute approximately 64.6% of total sales
•	Low Fat products perform comparatively better in Tier 1 outlets
•	Medium outlets generate the highest overall sales
•	Category-level behavior shows health-focused demand for Low Fat products
________________________________________
Recommendations:
•	Increase Low Fat product availability in Tier 1 outlets
•	Maintain strong Regular product presence in high-volume categories
•	Use Medium outlets for testing new health-oriented SKUs
________________________________________
Conclusion:
This project demonstrates how advanced Excel techniques can be used to deliver end-to-end business intelligence solutions, from raw data to executive-ready dashboards and actionable insights.
________________________________________
Tools Used:
•	MS Excel 
o	Formulas (SUMIFS, AVERAGEIFS, IF, LOOKUPs)
o	Pivot Tables & Pivot Charts
o	Slicers for interactivity
o	Dashboard layout & formatting
